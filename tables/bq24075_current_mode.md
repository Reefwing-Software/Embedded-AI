| EN2 | EN1 | Maximum input current into IN pin     |
|-----|-----|---------------------------------------|
|  0  |  0  | 100 mA, USB100 mode                   |
|  0  |  1  | 500 mA, USB500 mode                   |
|  1  |  0  | Set by an external resistor from ILIM to VSS |
|  1  |  1  | Standby (USB suspend mode)            |
